## 🔗 Deploy

## 📝 Como submeter um projeto
Preencha o formulário: [Link do Google Forms](https://forms.gle/EfhUCQAM9CbypPxt9)

OU

Se quiser um desafio extra, abra uma PR direta para o arquivo projetos.json com o modelo correto. 
Não esqueça de enviar também a imagem na pasta thumbs e coloque o mesmo nome no campo de thumb:

```JSON
{
  "id": 4,
  "nome": "João Silva - Exemplo", 
  "projeto": "Site Pessoal Muito Maneiro",
  "github": "https://github.com/joao/site-pessoal-maneiro",
  "video": "https://www.youtube.com.....", 
  "thumb": "thumbs/nome_da_sua_img.jpg",
  "descricao": "Portfólio pessoal"
}
```
